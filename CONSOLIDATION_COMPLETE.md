## 🔍 WHAT IF THINGS GO WRONG?

### Service Won't Start
```bash
# Check what's using the port
netstat -ano | findstr :5000

# Kill the process
taskkill /F /PID <PID>

# Restart
start_services.bat
```

### SVG Diagrams Not Showing
```bash
# Check dependencies
pip install svgwrite

# Verify sequence format (A/U/G/C only)
# Restart web app
```

### Database Error
```bash
# Reset database
del web_app/instance/helix_zero.db

# Reinitialize
cd web_app && python -c "from app import app, db; app.app_context().push(); db.create_all()" && cd ..
```
